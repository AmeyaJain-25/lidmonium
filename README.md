# LidMonium

A **virtual harmonium** in your browser — **play by lid**: move your laptop lid like a harmonium bellow. Use your computer keyboard or the on-screen keys, with support for **Western notes** and **Indian sargam** notation, preset songs, and a configurable tonic.

**[→ GitHub](https://github.com/AmeyaJain-25/lidmonium)**

---

## Features

- **On-screen keyboard** — Click or use your computer keyboard to play
- **Preset songs** — Play along with built-in tunes; supports sargam (Indian notation) and Western notes
- **Configurable tonic** — Set the base note (e.g. C, D) for relative notation
- **Clean UI** — Simple layout with playlist, playback controls, and optional sargam label

## Scripts

| Command      | Description                     |
